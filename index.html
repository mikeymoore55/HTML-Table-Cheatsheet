HTML Tables Reference Sheet!

Table Tags

Tag Name Descripton
<table> Table Used to create tables for any sort of data best sorted in a table.
<thead> Table Head Used to section off the rows reserved for the table headings.
<tbody> Table Body Used to secion off the main portion of the table containing the information.
<tr> Table Row Used to create rows for our tables. The first step in adding data to the table.
<td> Table Data Each Table Row will contain Table Data elements. These Table Data elements are used to fill each cell of a table.
<tfoot> Table Foot Used to secion off the bottom portion of the table for totals and end notes.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.
